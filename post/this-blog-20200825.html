I just created this blog using a custom static site generator that I created using Kotlin and MVP CSS.

So, how does this blog work? It reads in markdown files containg the content from a git repository and from that, generates the HTML that you can see as a static site. So far this is really basic; it just takes this markdown but can't handle images or other embeded elements. That is something I want to try and include at a later date.
